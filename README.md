# projet-CDI


## Commandes GitHub utiles
- Initialiser le dépôt :
```bash
git init
git add .
git commit -m "Exemple"
git branch -m main
git remote add origin https://github.com/venghour5/projet-CDI
git push -u origin main
```
- Mettre à jour ton code :
````bash
git add .
git commit -m "Ajout de la classe Personnage (exemple)"
git push
````

- Récupérer les changements du code:
````bash
git pull origin main
````

- fusionner le code:
````bash
git checkout main
git merge --no-commit feature
